# Скрипты анализа логов сознания

Набор Python-скриптов для анализа JSON-логов системы процессуального сознания. Каждый скрипт анализирует определённый аспект работы системы.

## Использование

Все скрипты принимают путь к лог-файлу как первый аргумент. По умолчанию используется `logs/run_20260415_031715.json`.

```bash
# Из корня проекта:
python3 analysis_log_scripts/01_overview.py logs/run_XXXXXXXX_XXXXXX.json
```

## Скрипты

### 01_overview.py — Общий обзор
Базовая статистика лога: количество тиков, длительность, распределение событий по типам.

**Что узнаешь:** сколько длилась сессия, сколько было ошибок, какие типы событий доминируют.

```bash
python3 analysis_log_scripts/01_overview.py logs/run_20260415_031715.json
```

---

### 02_phase_timeline.py — Хронология фаз
Показывает фазы состояния системы по активным каналам гистерезиса. Фильтрует короткие фазы.

**Что узнаешь:** когда система была спокойна, когда в стрессе, когда в эйфории. Длительность каждой фазы в тиках и минутах.

```bash
# Показать фазы длиннее 10 тиков (по умолчанию):
python3 analysis_log_scripts/02_phase_timeline.py logs/run_20260415_031715.json

# Показать все фазы длиннее 5 тиков:
python3 analysis_log_scripts/02_phase_timeline.py logs/run_20260415_031715.json 5
```

---

### 03_thoughts_evolution.py — Эволюция мыслей
Выборка спонтанных мыслей и рефлексий через равные интервалы по времени. Показывает как менялось содержание мышления.

**Что узнаешь:** о чём система думала в начале, середине и конце сессии. Как менялась глубина и тональность.

```bash
python3 analysis_log_scripts/03_thoughts_evolution.py logs/run_20260415_031715.json
```

---

### 04_internal_stimuli.py — Внутренние стимулы
Анализ стимулов, которые рефлексия генерировала сама для обработки полным pipeline. Тематическая классификация.

**Что узнаешь:** о чём система хотела "подумать глубже", какие темы доминировали, есть ли зацикливание.

```bash
python3 analysis_log_scripts/04_internal_stimuli.py logs/run_20260415_031715.json
```

---

### 05_hysteresis_analysis.py — Анализ гистерезиса
Статистика стимуляции каналов: от Emotion-агента (внешняя) и от Reflection-агента (самостимуляция). Детектирует "спираль смерти".

**Что узнаешь:** какие каналы стимулировались чаще всего, пыталась ли система себя успокоить (отрицательная стимуляция), сколько времени система провела в залипшем состоянии.

```bash
python3 analysis_log_scripts/05_hysteresis_analysis.py logs/run_20260415_031715.json
```

---

### 06_runtime_deltas.py — Анализ runtime-дельт
Статистика изменений runtime-параметров: temperature, context_window, energy_level, attention_focus, latency, bandwidth. Включает данные о vitality и max_tokens.

**Что узнаешь:** как менялись "физические" параметры системы, какие параметры страдали больше всего, на каком уровне работал max_tokens.

```bash
python3 analysis_log_scripts/06_runtime_deltas.py logs/run_20260415_031715.json
```

---

### 07_responses_analysis.py — Анализ ответов
Статистика ответов системы: уникальность, повторения, длина по времени. Отдельно анализирует реакцию на внешние стимулы от пользователя.

**Что узнаешь:** насколько разнообразны ответы, есть ли зацикливание, как система реагировала на пользователя.

```bash
python3 analysis_log_scripts/07_responses_analysis.py logs/run_20260415_031715.json
```

## Полный прогон всех скриптов

```bash
for script in analysis_log_scripts/0*.py; do
    echo "=== $(basename $script) ==="
    python3 "$script" logs/YOUR_LOG_FILE.json
    echo
done
```
